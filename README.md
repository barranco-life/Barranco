# Proyecto Cultural Barranco

Barranco es un espacio cultural alternativo en Mallasa, La Paz: jardín, casa, quebrada, barra, fuego, música, cine, yoga, comida, conversación y comunidad.

Este repo empieza como una base pública y viva para documentar formas de habitar el espacio en **modo Barranco**: con libertad, confianza, cuidado y la intención de dejar el lugar igual o mejor.

## Encuentros

- [Cómo hacer tu encuentro en Barranco](./encuentros/README.md)

Esta primera versión no es una tarifa final ni un contrato. Es una forma de abrir conversación para encuentros, talleres, cumpleaños, comidas, cine, música, yoga u otras propuestas que tengan sentido con el espacio.
